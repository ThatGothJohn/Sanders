# Sanders
A multithreaded Kernel

#### required software (Ubuntu package names)
* make
* qemu-system-x86_64
* build-utils
* nasm
* git
* bison
* flex
* libgmp3-dev
* libmpc-dev
* libmpfr-dev
* texinfo
* libisl-dev

### Build and install a cross-compiler
* ```./build_and_install_cross_compiler.sh```

## Build and run
* ```./build.sh```
* ```make run```
